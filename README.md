# The Legist ⚖️

A full-stack legal-tech platform built during a 12-hour college hackathon to simplify the process of connecting clients with lawyers and managing legal cases through a centralized digital platform.

The platform enables clients to discover lawyers based on specialization, submit legal requests, track ongoing cases, and communicate efficiently while providing lawyers with a dedicated dashboard to manage consultations and case workflows.

---

## Features

- Secure Authentication
- Role-Based Access (Client / Lawyer)
- Lawyer Directory
- Case Filing
- Case Tracking Dashboard
- Client-Lawyer Matching
- Consultation Requests
- Responsive User Interface
- RESTful APIs
- MongoDB Database Integration
